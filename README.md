<h1>GuruDelivery</h1>
Repositorio do GuruDelivery um aplicativo(web) de entrega de comida para o projeto final da disciplina de DAC

<h2>Instruções</h2>
<p>• para rodar essa aplicação clone este repositorio; <br>
• ao clonar entre no diretorio raiz e abra o terminal nesta pasta <br>
• em seguida instale as seguintes dependencias: <br>
</p>

```python
pip install -r requirements.txt
```

<p>• Após isso rode o servidor django: <br>
<br>
</p>

```python
python manage.py runserver
```

<p>• Para ter acesso ao painel de administrador <br>
<br>
• Será necessario primeiro criar um super usuario da aplicação
</p>

```python
python manage.py createsuperuser
```

<p>• E Então poderá criar usuarios, hamburguers, novos admins etc <br>
<br>
</p>
